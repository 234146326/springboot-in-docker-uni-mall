# springboot-in-docker
 docker-compose spring-boot mysql nginx
 
 install:  
 1,clone code; 
```
git clone https://github.com/234146326/springboot-in-docker-uni-mall.git
```

 2,init project
 ```
./init.sh
```

 3,Docker install    
 ```
   docker-compose up -d 
```
